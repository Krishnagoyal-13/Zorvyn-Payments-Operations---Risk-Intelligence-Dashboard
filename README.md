# Zorvyn Payments Operations & Risk Intelligence Dashboard

Portfolio-grade Next.js dashboard simulating a fintech data analyst workflow:
- data cleaning and quality controls
- KPI formula logic
- aggregation and visualization
- risk and merchant monitoring
- forecast vs actual analysis

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:3000.

## Stack
- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Recharts
